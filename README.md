# Shiwei Hong — Academic Website

This repository is configured as the GitHub Pages user site for:

- GitHub account: `Shiwei-Hong`
- Repository: `Shiwei-Hong.github.io`
- Website: `https://shiwei-hong.github.io`

## Site structure

- `_pages/about.md`: homepage, research introduction, and news
- `_bibliography/papers.bib`: publication metadata
- `_data/socials.yml`: public contact and profile links
- `assets/img/prof_pic.jpg`: profile photo

## Deployment

1. Upload all files and folders in this directory to the repository root.
2. In **Settings → Actions → General**, set **Workflow permissions** to **Read and write permissions**.
3. Run the **Deploy site** workflow under **Actions**.
4. After the `gh-pages` branch is created, go to **Settings → Pages** and select **Deploy from a branch**, `gh-pages`, and `/ (root)`.

Edit source content on `main`. Do not edit the generated `gh-pages` branch.
